## 2. 风格特征 (8 条)

### 特征 1: **反问开场** — 14/20 (70%)

5 大 hook 模板 (按频次):

| Archetype | 例 | 占比 |
|---|---|---|
| 反问开场 | "X 能不能 Y?", "为什么 X?" | 70% |
| 假设开场 | "如果 X 会怎样?" | ~25% |
| 数字开场 | 较少, 多在反问后接 | <5% |
| 反常识开场 | "你以为 X, 其实 Y" | 5% |
| 场景/新闻开场 | "前两天, X 发生了..." | 5% |

**关键模式**: 反问 + 立即给答案 ("直接说答案, 因为...") — 7/20 在前 50 字内抛答案。

**代表原文**:
- #1: "因为核弹能不能抹平一座山, 比如泰山。**直接说答案, 不能**"
- #4: "人类的意志能不能扛得住酷刑**直接说答案**"
- #6: "为什么狼群不敢攻击有牧羊犬保护的羊群? **难道狼群还打不过一条狗吗?**" (反问 + 反问)

### 特征 2: **段子化自创梗** — 标志性 4 个

| 梗 | 出现视频 | 形式 |
|---|---|---|
| **夏侯惇鉴宝——一眼假** | #1, #2, #5, #20 (≥4 次) | 反差结构: 历史人物 + 现代动作 |
| **恐怖直立猿** (人类自指) | #3, #6, #7 (≥3 次) | 跨学科戏称, 配合游戏化叙事 |
| **地球online** (把地球当游戏) | #3 (4-5 次) | 跨学科比喻, 串联天体物理 + 进化 |
| **P2P / 一派胡言 / 一眼望不到** | #1, #2, #8, #14 | 反差降维 |

**段子化金句的 4 种结构**:
1. **X——Y 破折号反差**: "夏侯惇鉴宝——一眼假" / "武则天死老公——失去理智"
2. **重复对称**: "灵魂下班了, 肉体还在加班" / "数学: 数字×计算能力"
3. **跨作品引用**: "三体人: 三个太阳伺候一个行星" / "武则天" / "夏侯惇"
4. **游戏化/段位化**: "恐怖直立猿" / "持械恐怖直立猿" / "热武器恐怖直立猿" (递进 3 段)

### 特征 3: **数字密度 + 数学对比**

不止"列数字", 是**算给你看**:
- #1: "小男孩 4.5 万吨 / 大伊万 5 千万吨 → 量提升 3300 倍, 火球只提升 20 倍"
- #1: "1965 苏联 14 万吨 → 100 米深 430 米直径 → 484 万立方米 → 泰山 600 亿立方米 → **至少 12400 颗**"
- #14: "蓝鲸 5 公分皮肤 + 十几公分皮下脂肪 → 毒牙扎不进, 扎进被稀释"

**数学对比的 3 种模式**:
1. "A 倍 / 提升 X 倍" (倍率反差)
2. "A 是 B 的多少分之一" (比例反差)
3. "粗略算一下 → 至少 N 颗/N 元" (估算到具体)

### 特征 4: **跨学科融合** (100%)

每篇至少 3 个学科交织:

| 视频 | 主学科 | 跨入学科 |
|---|---|---|
| #1 核弹泰山 | 物理 (TNT, 火球) | 数学 (3300 倍), 地理 (板块运动), 历史 (1965 苏联) |
| #2 琥珀防腐 | 生物 (昆虫, 细菌) | 历史 (古埃及, 马王堆), 物理 (恒温高压), 化学 (水银) |
| #3 地磁暴 | 天体物理 | 游戏 (地球online), 进化 (蓝细菌), 历史 (广岛) |
| #4 酷刑 | 心理学 | 历史 (革命先烈), 生理 (疼痛阈值) |
| #14 耐毒王 | 生物 (毒液) | 游戏 (血条机制), 化学 (粘液瘤病毒) |

### 特征 5: **反转结构** — 3.9 反转词/视频

每 50s 一个反转, 模式:
- "但/不过/其实/真相是" 句首
- 4 层反常识: **反常识判断 → 段子化反差 → 数字支撑 → 数学对比**
- 例 #1: 反常识 (核弹不能抹平) → 段子 (夏侯惇鉴宝) → 数字 (火球 9.2 公里) → 数学 (12400 颗)

### 特征 6: **IP 签名收尾** — 16/20 (80%)

固定收尾: "**说出吾名, 吓汝一跳**" (Azure STT 偶识别错为"五名/五鸣/武明", 但原文是"吾名")

例外 (3 篇):
- #5 叙事收尾 (植入式广告结尾, 非主线)
- #18 合集广告结尾 ("关注一波绝对不亏")
- #20 **抒情治愈收尾** ("希望大家都能如愿睡个好觉") — **这是反例, 之前 style doc 已禁**

### 特征 7: **快语速 + 短节奏**

- **CPS 6.93** 远高于一般 TTS 5.0-5.4
- 单视频内部密度变化小 (avg_sent_len 40, short_ratio 29%)
- 标点密度高, 不断句=不断气, 听感紧凑

### 特征 8: **"咱们"叙述者视角**

- 第一人称"我"用得少 (偶尔出现 "我说", "荣说")
- 主体是"咱们"/"你"对话
- 自称"荣"/"行者道荣"作 IP 锚点, 但不当主语

## 3. 三大公式 (从 20 篇反推的模板)

### Hook 公式

```
[反问 X 能不能 Y?]
+ 立即给答案 ("直接说答案, 因为...")
+ 一句段子化反差 ("X——Y" 或 "就像 Z")
+ 第一个具体数字
```

**变体**:
- 假设型: "如果 X 会怎样? [立即给答案] 因为..."
- 反常识型: "你以为 X, 其实 Y" (用得少, 因为容易落入俗套)
- 双反问: "X? 难道 Y?" (#6 #3 风格)

### 中段钩子公式 (每 10-15s 一卡)

```
[具体数字 + 单位]
或 [段子化比喻 — 历史人物 + 现代动作]
或 [数学对比 — A 倍 / 提升 X]
或 [跨学科引用 — 三体/地球online/夏侯惇]
```

**钩子密度**: ~3-4 个/视频 = 每 50s 一个。

### 结尾公式 (80% 一致)

```
[最后一段: 把核心论点用"段子化金句"再砸一次]
+ (可选) "评论区 X" 唤起记忆/晒/站队
+ IP 签名 "说出吾名, 吓汝一跳"
```

**例外**: 植入广告 / 合集广告 不带 IP 签名。

## 4. 给我们的启示

### ✅ 能学 (1-2 周可上手)

1. **反问开场 + 立即给答案** — 任何主题都能套
2. **数字密度** (≥10/视频) — 每个论点配 1-2 个具体数字
3. **数学对比** (≥2/视频) — 不只列, 要算/比
4. **跨学科 2-3 个** — 同一论点串不同领域
5. **结尾 IP 签名** — 强化记忆点
6. **反转密度** (3+/视频) — "但是/其实/真相是" 句首
7. **快语速** — 6+ cps, 句短不断气

### ⚠️ 难学 (需长期积累)

1. **自创梗能力** ("夏侯惇鉴宝", "恐怖直立猿") — 需要创意 + 文化资本
2. **跨学科知识库** (物理+历史+数学+三体) — 8 年程序员可补, 但要 3-6 个月
3. **段子化节奏感** — 训练 6 个月+, 大量看 + 模仿
4. **快语速 TTS** — 我们 Radio_Host 默认 5.4 cps, 需调 speed 到 1.3 才能追上

### ❌ 不能抄

1. **"科普"赛道** — 红海, 你 8 年程序员切入没优势
2. **"夏侯惇/三国/三体" 等历史人物梗** — 你没积累, 用了 = 模仿
3. **"行者道荣" IP 名** — 你还没立人设
4. **100% 跨学科物理/生物** — 跟你的程序员/打工人调性不匹配

### 🎯 我们能落地的"3 个杠杆"

| 杠杆 | 怎么做 | 优先级 |
|---|---|---|
| **杠杆 1: 反常识判断** | "你以为 X, 其实 Y" 类开场, 任何主题都能套 | 高 |
| **杠杆 2: 数学对比** | "算给你看" 类, 程序员/财务/常识 都能算 | 高 |
| **杠杆 3: 自创程序员梗** | "这代码谁写的" "我司 X 同事" "Git log 里" | 中 |

## 5. 建议选题方向 (基于本号调性, 不是直接抄对标)

**调性**: 8 年程序员 + 公众号/视频号 + 变现, 偏打工人/职场/商业

| 方向 | 选题示例 | 套用对标的哪一条 |
|---|---|---|
| 程序员视角 | "为什么大厂 35 岁就优化你? 算下你的人力成本曲线" | 反常识 + 数学对比 + 数字 |
| 商业/职场 | "你每天点外卖, 平台到底抽多少? 算给你看" | 数学对比 + 段子化 (抽成链路) |
| 城市/生活 | "30 年房贷, 你实际付了多少钱? 算完惊了" | 反常识 + 数学 + 数字 |
| 认知/反常识 | "为什么彩票中 500 万你反而会变穷" | 反常识 + 数学 + 段子化 |

**核心**: 不做纯科普 (红海), 做"程序员/打工人视角 + 反常识 + 算给你看"。

## 6. 关键技术发现 (给 video-studio 工程用)

### CPS 差距

| 维度 | 对标实测 | video-studio 默认 | 缺口 |
|---|---|---|---|
| CPS | 6.93 | 5.4 (Radio_Host, speed 1.0) | **慢 22%** |
| 200s 字数 | 1400 | 1080 (200×5.4) | 缺 320 字 |
| 110s 字数 | 763 | 594 (110×5.4) | 缺 169 字 |

**对策**:
- `audio.speed = 1.3` 让 Radio_Host 跑到 7.0 cps (跟对标齐)
- 或写脚本时按 CPS 7.0 计算字数 (200s 目标 1400 字, 110s 目标 770 字)
- 当前 `MAX_SCRIPT_CHARS=1200` 已留余量, 200s 跑通

### Hook 公式样本 (给 LLM 学)

**4 条好的开场, 各覆盖一个 archetype**:

```text
[反问 + 立即给答案] (核弹泰山 #1, 65 字)
"核弹能不能抹平一座山, 比如泰山。直接说答案, 不能。咱们往往会过于高估核弹的威力。小时候经常听说把地球上所有的核弹一起引爆就能毁灭地球好几次。"

[假设 + 立即给答案] (琥珀防腐 #2, 50 字)
"既然琥珀里的昆虫万年不腐, 古人为什么不用琥珀来给尸体做防腐? 直接说答案, 因为琥珀它根本就不防腐。"

[反问 + 反问] (狼群牧羊犬 #6, 60 字)
"为什么狼群不敢攻击有牧羊犬保护的羊群? 难道狼群还打不过一条狗吗? 唉, 当然是因为牧羊犬有个终极技能叫做召唤恐怖直立猿。"

[假设 + 反常识] (熬夜 #20, 70 字)
"如果每天都熬夜到 12 点之后再睡觉会怎样? 熬夜的后果想必大家都知道, 每熬夜六十秒就会少睡一分钟。"
```

### 反模式 (给 LLM 避雷)

```text
❌ 抒情陈述开头: "凌晨两点, 灯还亮着" (v_40376759 反例)
❌ 自我重复: "熬的不是夜, 是 X" / "你 X 的不是 Y, 是 Z" (用了 2 次 = 没创意)
❌ 公众号爆款词堆砌: "KPI / 甲方 / 群消息 / 房东的租约" 5 个名词并排
❌ 烂大街金句: "早睡是奢侈品" / "成年人的崩溃"
❌ 治愈系结尾: "最后希望大家都能如愿睡个好觉" (#20 反例)
❌ 抒情: "把 X 揉进 Y 里" / "在 Z 里寻找 W"
```
