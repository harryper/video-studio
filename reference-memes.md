## 入选门槛 (4 条, 必须全过)

1. **钩子必须是该人物/梗广为人知的真实特征** (残疾 / 死亡 / 失败 / 历史典故)
2. **必须双关或同音字 PUN** (一个意思挂历史, 一个意思挂现代)
3. **必须配 [xingzhe] 段子化金句的 4 种结构之一** (见 §4)
4. **不只"古人+现代活动"空贴, 必须服务于脚本主题**

## 段子化金句的 4 种结构 (从 20 篇反推, 必背)

### 1. X——Y 破折号反差 (主力, ≥50% 出现率)
- 段子: "夏侯惇鉴宝——一眼假"
- 模板: `[历史人物/常识A]——[反差结论B]`
- 范例: "你买的房——其实是 30 年劳动期货" / "你以为在省钱——其实在给银行打工"

### 2. 重复对称 (≥10% 出现率)
- 段子: "灵魂下班了, 肉体还在加班"
- 模板: `[A 状态] 了, [B 状态] 还在 [动作]`
- 范例: "数据算完了, 用户还没问" / "钱没省下来, 时间花光了"

### 3. 跨作品引用 (≥15% 出现率)
- 段子: "三体人: 三个太阳伺候一个行星" / "武则天" / "夏侯惇"
- 模板: `[其他作品人物/概念], [套用到当前主题]`
- 范例: "三体人觉得这个算法很暴力" / "吕布跳槽, 三家都没要"

### 4. 游戏化/段位化 (≥10% 出现率, 必递进 3 段)
- 段子: "恐怖直立猿 → 持械恐怖直立猿 → 热武器恐怖直立猿"
- 模板: `[基础形态] → [升级 1] → [升级 2]`
- 范例: "普通员工 → 中层管理 → 高管 (其实都是同一个人)"

## 调性边界

- **准**: 反派 (盖章坏人) + 戏说古人 (非正史英雄) + 段子化自创 (地球online 这种)
- **不准**: 悲剧英雄 (项羽/岳飞/关羽) + 革命先烈 + 受害者
- **不准**: 治愈/松弛/愿你/希望 (style doc 已有, 引用即可)
- **不准**: 抒情 ("把 X 揉进 Y 里" / "在 Z 里寻找 W") — 烂大街金句
- **不准**: 公众号爆款词堆砌 (KPI/甲方/群消息/房东的租约 5 个名词并排)
- **不准**: "熬的不是夜, 是 X" / "你 X 的不是 Y, 是 Z" — 用了 2 次 = 没创意

## 维护纪律

- **不要堆数量**: 9 条已满, 50 条水货稀释 LLM 注意力
- **看到弱段子就删**: 直接编辑 markdown 表格删, 不留次品
- **种子是稀有资源**: 加新人物 = 严重决定, 必须能举出 1 个"同音字 PUN" + 1 个"现代场景" + 1 个"在 [xingzhe] benchmark 出现过" 才加
- **格式多样性**: 看到好 Q&A / 4 字 meme barrage 立刻加, 增加多样性
- **不脱离 [xingzhe] 风格**: 新段子必须能在 20 篇 transcript 里找到使用先例, 否则不算 [xingzhe] 段子
