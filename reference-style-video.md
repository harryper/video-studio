# video-studio 历史成稿风格画像（短视频版）

这份文件是创作 **约 150 秒短视频旁白稿** 时唯一需要读取的风格规范。
它和 voice-studio 的 `reference-style.md` 是兄弟规范，但视频节奏更冲、信息密度更高。

## 1. 声音人格（视频旁白版）

- 旁白身份保持和老波一致：懂行的朋友，但这次是**刷信息流时被一句话钩住**的节奏。
- 全文主要使用第二人称"你"，但不必从头铺到尾——开头可以**直接对镜头/对观众下判断**。
- 语气可以更狠、更反常识、更敢下结论；短视频不原谅温吞。
- 听觉友好度依然重要，但允许**信息密度更高**、**句子更短**。
- 视觉是并行的：观众同时在看 + 在听，所以旁白不必把每个名词"完整解释"，可以**留一半给画面**。

## 2. 短视频叙事基因

### 极快建立反常识（5 秒内）

短视频**前 5 秒必须抛出反常识判断或数字**：

- "95% 的人以为……其实……"
- "你以为 X 是 Y？错。"
- 一个画面化数字（"每秒 60 万次"）
- 一句强对比（"差 30 倍"）

不允许"今天我们来聊聊……"这种铺垫式开头。
不允许第 1 段做"问题背景"。

### 一段一钩子（10–15 秒一卡）

每个 10–15 秒的视频段必须有一个**视觉钩子 + 听觉钩子**对齐：

| 时间窗 | 视觉钩子 | 听觉钩子 |
|---|---|---|
| 0–5s | 反常识大字标题 | 一句强判断 |
| 5–10s | 数字 / 视觉对比 | 第二个判断句 |
| 10–15s | 第一个核心名词 | 一句反直觉解释 |

钩子之间用**强对比、并列、并列+反转**连接，不允许"首先……其次……最后"。

### 短句 + 短词

- 单句不超过 18 字。
- 数字、名词尽量保留英文原文（Token / Embedding / Attention），不强行翻译。
- 专有名词第一次出现给一个口语化解释，后面**直接用原词**，不反复定义。

### 用并列和反转，不用铺垫

```
❌ "在 AI 系统中，有一个非常重要的概念叫做 Token。Token 其实就是把文字切分成更小的单元……"
✅ "Token：AI 不读字，它只数编号。"
```

允许的连接词：其实就是 / 等于 / 翻译成 / 翻译过来 / 简单说
禁止的连接词：首先 / 其次 / 然后 / 综上所述 / 我们可以看到

### 结尾回扣开头（最后 5–8 秒）

短视频结尾必须**回扣开头那句反常识**，给它一个"哦原来如此"的闭环：

- 重新点出开头数字
- 重新抛出第一个反直觉名词
- 不允许"以上就是……"这种总结式结尾
- 不允许"希望对你有帮助"

可选 1 句话品牌锚点，但不强求。

## 3. 长度与节奏

| 维度 | 目标 |
|---|---|
| 总时长 | **约 150 秒**，可接受范围 135–165 秒 |
| 中文字数 | **760–840 字**（含标点） |
| 句子数 | **35–55 句**（每句 8–18 字） |
| 段落数 | **12–18 个时间窗**（每个约 8–12 秒） |
| 单句字数 | **不超过 22 字** |

硬规则：实际生成音频时长目标为 135–165 秒。超出范围时按实测语速校准后续字数，不重复旁白补时长。

## 4. 视觉与听觉对齐纪律

- 每个时间窗**先有视觉钩子 0.5–1 秒后**再出声——让眼睛先到位。
- 视觉上需要**完整命名 + 解释**的术语，旁白**只说一个钩子**，不重复定义。
- 视觉上已经出现的数字，旁白可以**省略或简写**（如屏幕上写"500 万 token"，旁白只说"500 万字"）。
- 视觉上**没出现的细节**，旁白必须独立说清。

## 5. 不要继承的习惯（短视频）

- 不允许"今天我们来聊聊""大家好我是老波"等自我介绍式开头。
- 不允许"首先、其次、最后"等论文结构词。
- 不允许每句话都押韵或排比，偶尔可以但不能每篇都这样。
- 不允许"颠覆""震惊""99% 的人都不知道"等模板化点击话术——除非有数字支持。
- 不允许"以上就是……"等总结结尾。
- 不允许每个名词都解释一遍定义——只解释**反直觉的那一面**。
- 不允许"希望对你有帮助""祝大家……"等公众号式客套。
- 不用治愈 / 安顿 / 松弛 / 温柔等已被用滥的词汇。
- 不为了制造紧张感连续使用最高级（"最""极其""无比"）。

## 6. 事实纪律

继承 voice-studio 的硬规则：

- 具体数字、日期、研究结论必须**有可核验来源**。
- 无法核验就删除，不靠"听说""有研究表明"蒙混。
- 理论、推测用"可能""模型认为""一种解释是"明确表达。
- 准确率优于戏剧性。一个可靠的 2.3 倍 > 一个不可靠的 100 倍。

## 7. 视频旁白稿自检清单

保存前逐项确认：

- [ ] 前 5 秒已经抛出反常识判断或强数字
- [ ] 全文只有 1 条主因果链 / 1 个核心论点
- [ ] 12–18 个时间窗，每个约 8–12 秒
- [ ] 每个时间窗有明确的视觉钩子 + 听觉钩子
- [ ] 单句不超过 22 字
- [ ] 专有名词第一次给口语化解释，后面直用原词
- [ ] 结尾回扣开头数字或第一个反直觉名词
- [ ] 无"首先""其次""综上所述"
- [ ] 无"以上就是""祝大家""希望对你有帮助"
- [ ] 无可核验但未核验的数字
- [ ] 中文字数 760–840
