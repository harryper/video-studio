# video-studio 短视频旁白稿风格画像 — [xingzhe] 风格对标

> 这份文件是创作 **60-200s 短视频旁白稿** 时唯一需要读取的风格规范.
> 数据基线: `benchmarks/xingzhe/analysis.md` (20 篇顶部视频反推, 1.8M 粉抖音号 [行者道荣]).
> 跟 `reference-memes.md` 配套使用: 本文件管结构 + 节奏, 那份管段子.

## 1. 声音人格

- **咱们 视角**: 第一人称"我"很少, 主体是"咱们"/"你"对话. 像"刷信息流时被一句话钩住的朋友".
- **语气**: 狠 / 反常识 / 敢下结论. 短视频不原谅温吞.
- **听觉友好**: 句子短, 不断气 = 不断句; CPS 6.0-7.0 (TTS speed 1.15-1.30).
- **视觉是并行的**: 旁白不必把每个名词"完整解释", 留一半给画面.

## 2. Hook 公式 (4 种 archetype, 前 5 秒必须落地一种)

### Archetype A: 反问 + 立即给答案 (主力, 70%)

```
[X 能不能 Y? 直接说答案, Z。]
+ 段子化反差 ("X——Y" 或 "就像 Z")
+ 第一个具体数字
```

范例 (rank_01 核弹泰山, 65 字):
> "核弹能不能抹平一座山, 比如泰山。**直接说答案, 不能**。咱们往往会过于高估核弹的威力。"

### Archetype B: 假设 + 立即给答案 (~25%)

```
[如果 X 会怎样? 直接说答案, Y。]
+ 反常识支撑
```

范例 (rank_02 琥珀防腐, 50 字):
> "既然琥珀里的昆虫万年不腐, 古人为什么不用琥珀来给尸体做防腐? **直接说答案, 因为琥珀它根本就不防腐。**"

### Archetype C: 反问 + 反问 (~5%)

```
[X 怎么样? 难道 Y 吗?]
+ 段子化反差给出"真正原因"
```

范例 (rank_06 狼群, 60 字):
> "为什么狼群不敢攻击有牧羊犬保护的羊群? **难道狼群还打不过一条狗吗? 唉, 当然是因为牧羊犬有个终极技能叫做召唤恐怖直立猿。**"

### Archetype D: 反常识开场 (5%, 慎用易俗套)

```
[你以为 X, 其实 Y。] (与 v_bench_new01 风格一致)
```

范例 (rank_20 熬夜, 70 字):
> "如果每天都熬夜到 12 点之后再睡觉会怎样? 熬夜的后果想必大家都知道, **每熬夜六十秒就会少睡一分钟**。"

**禁用开场** (反模式):
- ❌ "今天我们来聊聊" 铺垫式
- ❌ "大家好我是 X" 自我介绍
- ❌ 抒情陈述 ("凌晨两点, 灯还亮着")
- ❌ 第 1 段做"问题背景"

## 3. 中段钩子公式 (10-15s 一卡, 3-4 个/视频 = 每 50s 一个)

每个 10-15s 视频段必须有 1 个**视觉钩子 + 听觉钩子**对齐, 钩子类型从下面 4 选:

1. **具体数字 + 单位**: "4.5 万吨", "9.2 公里", "227 克" — 数字必须配单位
2. **段子化比喻** (历史人物 + 现代动作): "夏侯惇鉴宝", "武则天死老公", "恐怖直立猿"
3. **数学对比**: "A 倍" / "提升 X 倍" / "A 是 B 的 1/N" / "粗略算一下 → 至少 N 颗"
4. **跨学科引用**: "三体人", "地球online", "持械恐怖直立猿"

**密度要求**: 数字密度 **≥10 个/视频** (含中+阿), 数学对比 **≥2 个/视频**, 反转词 ("但是/其实/真相是") **≥3 个/视频**.

**钩子连接**: 用**强对比、并列、并列+反转**连接, 不允许"首先……其次……最后".

## 4. 编号结构 (v_bench_new 风格, 我们的强项)

主题偏职场/经济/常识时, 强制用**编号**展开. 编号前必须有 1 句 **setup** 解释"为什么用编号" (避免突兀), 编号段直接陈述本段主题, **不在编号开场用反转**.

**Setup 模板** (1 句话, 10-25 字, 必须有):
- "**咱们先算 [N] 笔账**, 看 [主题] 怎么 X 的."
- "**这事分 [N] 层看**, 每层都不简单."
- "**这场战争分 [N] 波打**, [主体] 一次比一次狠."
- "**这个事儿我能给你拆 [N] 笔**, 听完你就懂了."

**3 种编号选型** (按主题):

- **第一笔 / 第二笔 / 第三笔** (v_bench_new01 HR + v_bench_new03 房贷) → 经济/职场/历史账目
- **第一层 / 第二层 / 第三层** (v_bench_new02 外卖) → 系统/机制/制度
- **第一波 / 第二波 / 第三波** (rank_06 狼群) → 攻击/演化/时间线

**每笔/层/波开场格式**:
- ✓ 1 句话**直接陈述本段主题** ("第一笔, 25 岁的 P5" / "第一笔, 雨." / "第一笔, 基层市集.")
- ✓ **不要在开场用反转** ("第一笔, 主角不是海, 是雨." ❌ — 反转浪费钩子, 让段内开场突兀)
- ✓ **不要在开场用抽象名词** ("第一笔, 来源." ❌ — 不够具体)
- ✓ 然后 80-150 字展开 (用具体数字 + 数学对比 + 段子化比喻)

**反模式**:
- ❌ Hook → 直接跳 "第一笔" (无 setup, 突兀)
- ❌ "第一笔, 主角不是 X, 是 Y" (反转在开场用 = 浪费段中段尾的反转位)
- ❌ "第一笔, 来源." (抽象名词, 不具体)

**完整范例** (海水为什么是咸的):
> Hook: "海水为什么是咸的? 直接说答案, 因为地球online 开了 45 亿年的服..."
> Setup: "**这事咱们拆三笔算**, 从天上到海底全过一遍."
> 第一笔, 雨. 雨水落下来, ... (80-150 字)
> 第二笔, 海面. 海面蒸发, ... (80-150 字)
> 第三笔, 海底. 海底火山口往外喷, ... (80-150 字)
> 结尾: "...你喝的是 45 亿年的地质账单."

## 5. 段子化金句 4 种结构 (必背, 来自 reference-memes.md §4)

1. **X——Y 破折号反差** (主力): "夏侯惇鉴宝——一眼假" / "你买的房——其实是 30 年劳动期货"
2. **重复对称**: "灵魂下班了, 肉体还在加班"
3. **跨作品引用**: "三体人", "武则天", "吕布跳槽, 三家都没要"
4. **游戏化/段位化** (必递进 3 段): "恐怖直立猿 → 持械 → 热武器"

**密度**: 1-3 段子/视频 (主题契合才用, 不强求). 每条段子必须服务脚本主题, 严禁末尾/开头/中间任意位置单独加 PUN 段子作为"冷知识/彩蛋/bonus".

## 6. 结尾公式 (80% 一致)

```
[最后一段: 把核心论点用"段子化金句"再砸一次]
+ (可选) "评论区 X" 唤起记忆/晒/站队
+ (可选) IP 签名 / 强反常识金句
```

**禁用结尾** (反模式):
- ❌ "以上就是……" 总结式
- ❌ "希望对你有帮助" 客套
- ❌ 治愈系 "希望大家都能 X" (#20 行者道荣自反例)
- ❌ 抒情 "把 X 揉进 Y 里"

**IP 签名**: [行者道荣] 用"说出吾名, 吓汝一跳" (16/20 = 80%). 我们 8 年程序员调性, **不抄这句**, 改用强反常识金句收尾 (例: "你以为你在买房, 其实你在给银行打 30 年工").

## 7. 长度与节奏

| 时长目标 | 字数 (CPS 6.0-7.0) | 段落数 | 段子数 (软目标) |
|---|---|---|---|
| 30s | 200-260 | 5-7 | 0-1 |
| 60s | 380-500 | 9-12 | 0-2 |
| 110s | 700-870 | 15-20 | 1-3 |
| 200s | 1280-1500 | 28-34 | 2-4 |

**单句**: ≤18 字 (短句率 ≥29%).

**CPS 校准**: 默认 TTS 5.4, 需 `audio.speed = 1.15-1.30` 跑到 6.2-7.0.

## 8. 视觉与听觉对齐

- 每个时间窗**先有视觉钩子 0.5-1s 后**再出声.
- 视觉已出现的数字, 旁白可省略或简写.
- 视觉上**没出现的细节**, 旁白必须独立说清.

## 9. 调性边界 (跟 reference-memes.md 同步)

- **准**: 反派 (盖章坏人) + 戏说古人 (非正史英雄) + 段子化自创 (地球online)
- **不准**: 悲剧英雄 + 革命先烈 + 受害者 + 治愈/松弛/愿你/希望
- **不准**: 抒情 ("把 X 揉进 Y 里")
- **不准**: 公众号爆款词堆砌 (KPI/甲方/群消息/房东的租约 5 个名词并排)
- **不准**: "熬的不是夜, 是 X" — 用了 2 次 = 没创意

## 10. 事实纪律

- 具体数字、日期、研究结论**必须有可核验来源**, 否则删除.
- 理论、推测用"可能""模型认为""一种解释是"明确表达.
- 准确率优于戏剧性. 一个可靠的 2.3 倍 > 一个不可靠的 100 倍.

## 11. 反模式 (20 篇反推 + v_bench_new 验证)

| 反模式 | 来源 |
|---|---|
| ❌ "今天我们来聊聊" 铺垫 | 普通抖音 |
| ❌ "大家好我是 X" 自介 | 普通抖音 |
| ❌ "以上就是……" 总结 | 烂大街 |
| ❌ "希望对你有帮助" 客套 | 烂大街 |
| ❌ 抒情 "把 X 揉进 Y 里" | 烂大街金句 |
| ❌ "熬的不是夜, 是 X" 重复 | 用 2 次 = 没创意 |
| ❌ 公众号爆款词堆砌 (KPI/甲方/群消息) | 营销文 |
| ❌ 治愈系 "希望大家都能 X" | #20 反例 |
| ❌ "颠覆""震惊""99% 的人都不知道" 模板化点击话术 | 除非有数字支持 |
| ❌ 每个名词都解释一遍定义 | 视觉已经讲了 |
| ❌ 抒情陈述开头 ("凌晨两点, 灯还亮着") | 自反例 |
| ❌ 自我重复 (X——Y 用 2 次以上) | 没创意 |

## 12. 视频旁白稿自检清单

保存前逐项确认:

- [ ] 前 5 秒已用 4 种 Hook 之一 (A 反问+立即给答案 / B 假设+立即给答案 / C 反问+反问 / D 反常识)
- [ ] 编号结构使用 (第一笔/第一层/第一波, 至少 2 笔/层/波)
- [ ] 数字密度 ≥10/视频, 数学对比 ≥2/视频
- [ ] 段子化金句 1-3 条, 4 种结构中至少用 1 种
- [ ] 段子服务脚本主题, 无末尾/开头"冷知识"包装
- [ ] 反转词 (但是/其实/真相是) ≥3/视频
- [ ] 单句 ≤18 字, 短句率 ≥29%
- [ ] 无禁用开场 (铺垫/自介/抒情)
- [ ] 无禁用结尾 (总结/客套/治愈)
- [ ] 调性边界守住 (无悲剧/革命/治愈)
- [ ] 字数匹配时长 (30s/200-260, 60s/380-500, 110s/700-870, 200s/1280-1500)
